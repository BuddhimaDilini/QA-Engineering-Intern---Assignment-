# Safora Contact Form Automation Test

This project contains a simple automation test script for the Safora Contact Us form using Selenium and Python.

## Features

- Opens the Safora website
- Navigates to the Contact Us page
- Tests form validation
- Verifies required field behavior
- Uses Selenium WebDriver with Firefox

## Technologies Used

- Python
- Selenium
- Firefox WebDriver (GeckoDriver)

## Installation

Install Selenium:

```bash
pip install selenium
```

Download GeckoDriver:
- https://github.com/mozilla/geckodriver/releases

## Run the Test

```bash
python test.py
```

## Test Scenario

1. Open Safora website
2. Navigate to Contact Us page
3. Attempt form submission without inputs
4. Verify validation messages are displayed
